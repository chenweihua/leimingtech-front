如果需要引用h5,api模块需要在pom.xml中放开注释的代码，放开代码如下：
<!-- <dependency>
	<groupId>com.leimingtech</groupId>
	<artifactId>leimingtech-h5</artifactId>
	<version>1.1-SNAPSHOT</version>
</dependency> -->

<!-- <dependency>
	<groupId>com.leimingtech</groupId>
	<artifactId>leimingtech-api</artifactId>
	<version>1.1-SNAPSHOT</version>
</dependency> -->
